# Features

1. Register/Login (one time)

- Teacher: Department, Year, Division, Lecture/Lab, Teacher code
- Student: Department, Year, Division, Lecture/Lab

2. Select Lecture/Lab

- select channel to join

3. Give/Take attendance press button to start attendance session

- Teacher: Press button to start attendance session
- Student: Press button to mark present

4. Receive Conformation

- Student receives conformation

5. Edit Info Manually

- Teacher can edit attendance manually if any technical issues arises

6. Export to Excel
